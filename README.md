下载之后需要在本地运行
```bash
npm install
```
才可以运行

## hexo常用命令

### 清除已经生成的public中内容
hexo clean

### 生成页面
hexo g

### 启动预览
hexo s

### 创建新的页面
hexo new page "new_page_name"

### 创建新的文章
hexo new "new post"
